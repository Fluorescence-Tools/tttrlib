Detailed Description

Run all plot_*.py examples in separate subprocesses with a timeout.
Write failing basenames to a blacklist file usable by conf.py.

- Cross-platform (Windows/macOS/Linux)
- Forces headless MPL (MPLBACKEND=Agg)
- Sets TTTRLIB_DOCS=1 so examples can downscale work when building docs
- "Fail" = non-zero exit OR timeout
- By default, removes files from the blacklist when they start succeeding (prune)
