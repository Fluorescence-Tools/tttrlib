        "\n# Intensity traces\nThe function ``tttrlib.intensity_trace`` can be applied to a sequences of event\ntimes traces to calculate the time-dependent counts for a given time window size.\nIn single-molecule FRET experiments such traces are particularly useful to illustrate\nsingle-molecule bursts. The function takes an array of macro times as an input\nthe macro times are binned and the counts per bin are returned.\n\n#.. plot:: ../examples/single_molecule/single_molecule_mcs.py\n\nAbove, for a donor and acceptor detection channel a histogram trace for is shown\nin green and red.\n\nThere are two options to compute intensity traces. Intensity traces can either be\ncomputed using the function ``tttrlib.compute_intensity_trace`` or using the method\n``intensity_trace`` of a ``TTTR`` object.\n\n```python\ntttr_object = tttrlib.TTTR(str(DATA_ROOT / 'bh/bh_spc132.spc'), 'SPC-130')\n# option 1\ntttr_object = tttrlib.compute_intensity_trace(\n    data.macro_times,\n    time_window_length=1.0, # this is one millisecond\n    macro_time_resolution=data.get_header().macro_time_resolution / 1e6\n)\n# option 2\ntttr_object = data.intensity_trace(\n    time_window_length=1.0, # this is one millisecond\n)\n```\nWhen the first option is used, a calibration for the macro time array needs to be\nprovided. If the second option is used, the macro time calibration of the header\ninformation that was processes from the tttr file when the ``TTTR`` object was\ncreated is already considered. The time window length are in units of milli seconds.\n"
