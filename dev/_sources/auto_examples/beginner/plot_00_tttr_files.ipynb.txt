        "\n# Basic operations with TTTR files\n\nThis example shows how to perform basic operations with TTTR files.\n\n# Introduction to Time-Tagged Time-Resolved (TTTR) Data with tttrlib\n\nTTTR data is a format used in single-molecule fluorescence, fluorescence lifetime imaging (FLIM), and time-correlated single-photon counting (TCSPC) experiments. TTTR files record streams of photon detection events, each with precise timing and detector channel information.\n\nEach event typically contains:\n  1. Macro time (experiment time or sync pulse number)\n  2. Micro time (photon delay after excitation pulse)\n  3. Channel number (detector ID)\n  4. Event type (photon, marker, etc.)\n\nThis rich, raw data enables advanced analyses such as burst detection, fluorescence lifetime calculation, FRET efficiency, and more.\n\nCommon Use Cases:\n  - Single-molecule FRET (smFRET) burst analysis\n  - Fluorescence Correlation Spectroscopy (FCS)\n  - Fluorescence Lifetime Imaging (FLIM)\n  - Photon counting histograms\n  - Custom photon stream filtering and selection\n\nInstallation & Setup:\n  You can install tttrlib using pip:\n      pip install tttrlib\n\n  Or, for the latest development version, clone the repository and install:\n      git clone https://github.com/fluorescence-tools/tttrlib.git\n      cd tttrlib\n      pip install .\n\n  You will need Python (>=3.7), NumPy, and optionally Jupyter for running notebooks.\n\nThe operations demonstrated in this script include:\n  1.
