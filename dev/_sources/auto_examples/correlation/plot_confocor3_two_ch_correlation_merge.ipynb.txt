        "\n# Confocor3 two channel cross-correlation (using merge method)\n\nThe raw FCS data format of the Zeiss Confocor3 is relatively simple.\nZeiss Confocor3 raw files store time-difference between photons.\nA relatively small header is followed by a set of 32-bit integers that\ncontain the time difference to the previous registered photon. Photons\nregistered by different channels are stored in separate files.\n\nThis example demonstrates how to use the new TTTR merge method to combine\nmultiple Confocor3 channels into a single photon stream for cross-correlation\nanalysis. The merge method automatically handles time sorting and is more\nefficient than manual concatenation.\n"
