        "\n# Polarization resolved lifetime MLE\n\n## fit23\n\nThis example illustrates how to recover a fluorescence lifetime (tau) and a\nrotational correlation time (rho)  using ``fit23`` from polarization resolved\nfluorescence decay histograms. ``fit23`` makes use of a maximum likelihood estimator\n(MLE) and thus can be applied to low photon count data as frequently encountered\nin time-resolved single-molecule spectroscopy or fluorescence lifetime imaging (FLIM).\n\n``fit23`` optimizes a single rotational correlation time $\rho$ and a\nfluorescence lifetime $   au$ to a polarization resolved fluorescence decay\nconsidering the fraction of scattered light and instrument response function in\nthe two detection channels for the parallel and perpendicular fluorescence. Fit23\noperates on fluorescence decays in the :term:`Jordi-format`.\n\n``fit23`` is intended to be used for data with very few photons, e.g. for pixel analysis\nin fluorescence lifetime image microscopy (FLIM) or for single-molecule spectroscopy.\nThe fit implements a maximum likelihood estimator as previously described :cite:`maus_experimental_2001`.\nBriefly, the MLE fit quality parameter 2I* = $-2\\ln L(n,g)$ (where $L$\nis the likelihood function, $n$ are the experimental counts, and $g$\nis the model function) is minimized. The model function $g$ under magic-angle\nis given by:\n\n$g_i=N_g \\left[ (1-\\gamma) \frac{irf_i \u0007st \\exp(iT/k       au) + c}{\\sum_{i=0}^{k}irf_i \u0007st \\exp(iT/k      au) + c} + \\gamma \frac{bg_i}{\\sum_i^{k} bg_i} \right]$\n\n$N_e$ is not a fitting parameter but set to the experimental number of\nphotons $N$, $\u0007st$ is the convolution operation, $    au$ is the\nfluorescence lifetime, $irf$ is the instrument response function, $i$\nis the channel number, $bg_i$ is the background count in the channel $i$,\n\nThe convolution by fit23 is computed recursively and accounts for high repetition\nrates:\n\n$irf_i \u0007st \\exp(iT/k      au) = \\sum_{j=1}^{min(i,l)}irf_j\\exp(-(i-j)T/k  au) + \\sum_{j=i+1}^{k}irf_j\\exp(-(i+k-j)T/k     au) $\n\nThe anisotropy treated as previously described :cite:`schaffer_identification_1999`.\nThe correction factors needed for a correct anisotropy used by ``fit2x`` are\ndefined in the glossary (:term:`Anisotropy`).\n"
