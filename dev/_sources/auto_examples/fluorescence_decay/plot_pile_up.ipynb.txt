        "import pylab as p\nimport numpy as np\nimport scipy.stats\nimport tttrlib\n\n# Set seed to make results reproducible\nnp.random.seed(42)\n\n# Define number of channels and excitation period\nn_channels = 512\nperiod = 32\ntime_axis = np.linspace(0, period, n_channels)\nconv_stop = n_channels // 2\n\n# compute a irf\nirf_width = 0.25\nirf_position = 2.0\ndt = time_axis[1] - time_axis[0]\nirf_p = scipy.stats.norm.pdf(time_axis, loc=irf_position, scale=irf_width)\nirf_s = irf_p\n\n\"\"\"\nIn this example first an ideal fluorescence decay is computed for the parallel (p)\nand perpendicular (s) channel using fit23.\n\"\"\"\n\n# Compute model fo a single photon\nn_photons = 1\nperiod, g, l1, l2 = 32, 1.0, 0.1, 0.1\ntau, gamma, r0, rho = 2.0, 0.01, 0.38, 1.2\n\nmodel = np.zeros(n_channels * 2)  # For parallel and perpendicualr\nbg = np.zeros_like(model)\ntttrlib.DecayFit23.modelf(\n    np.array([tau, gamma, r0, rho]),\n    np.hstack([irf_p, irf_s]),\n    bg, dt,\n    np.array([period, g, l1, l2, conv_stop]),\n    model\n)\n\n\"\"\"\nThe number the model is scaled to a number of photons typically recorded in a eTCSPC \nexperiment.
