        "\n# Make a 1D Histogram\n\n``tttrlib`` offers some functions and classes for the creation of histograms.\nThey are part of ``tttrlib`` to offer more performance in direct comparison to\ngeneral purpose routines for analysis methods as Photon Distribution Analysis\n(PDA) that make heady use of repeated histogram calculations.\n\n<div class=\"alert alert-info\"><h4>Note</h4><p>These function and ``classes`` are not intended as a substitution for more\n    general functions, e.g., offered by [NumPy](https://www.numpy.org/) and\n    are less tested. They offer performance advantages for histograms\n    calculated on a logarithmic (log10) scale and a linear scale in direct\n    comparison to NumPy.</p></div>\n\n## One dimensional histograms\n\n``tttrlib`` supports one dimensional histograms with linear, logarithmic, or\narbitrary spaced bins.\n\n#.. plot:: ../examples/miscellaneous/histogram_1D.py\n   :include-source:\n\nIn the example shown above, the histograms for the three bin spacings are shown.\n\n\n## Multi dimensional histograms\n\nIn addition to simple one dimensional histograms ``tttrlib`` can calculate\nmultidimensional histograms with an arbitrary number of dimensions. For that,\nfirst a new object of the class ``doubleHistogram`` is created. Here, *double*\nrefers to the data type double - a floating point number. Next, axes are added\nto the Histogram by calling the method ``setAxis``. The first argument is the\nnumber of the axis, the second the name of the axis, the third and the fourth\nthe range of the axis, the fifth the number of bins, and the sixth the type of\nthe axis. The ``Histogram`` class support arbitrary axis types. However, when\ninitializing an axis via ``setAxis`` the axis type must be either *lin* or\n*log10* for a linear or an logarithmic axis with base 10. Next, the data are\nassigned to the histogram by the ``setData`` method and the histogram is\ncalculated by the ``update`` method. A working example for 2D normal\ndistributed data is shown below.\n\n#.. plot:: ../examples/miscellaneous/histogram_2D.py\n   :include-source:\n\nAbove, a two dimensional histogram with linear spaced bins is shown.\n"
