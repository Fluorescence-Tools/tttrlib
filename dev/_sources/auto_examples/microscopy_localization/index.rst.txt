Image Localization Examples
============================

Examples demonstrating single molecule localization using 2D Gaussian fitting.

These examples show how to use the ImageLocalizer class to precisely localize
fluorescent point sources (e.g., fluorescent beads or single molecules) with
sub-pixel accuracy using 2D Gaussian curve fitting.
