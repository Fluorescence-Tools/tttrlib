uint64_t bh_spc600_4096_record::allbits

The information about the subsequent photons is stored one after another in the measurement data file. For each photon 6 bytes are used. The parameter

Parameters
adccorresponds to the value of the analog to digital converter and related to the micro time (mt) by mt = (4095 - adc) * TACRange / (TACGain * 4096)
invalidis true for invalid TTTR records. If invalid is true all data in the record except the mtov bit are invalid.
mtovmarks that a macro timer overflows. In case
mtovis true 2**24 counts should be added to the overflow counter (the overflow counter counts the number of overflows.
gapmarks possible gaps, e.g., due to FIFO overflows.
routprovides the (inverted) routing number of the TTTR record.
